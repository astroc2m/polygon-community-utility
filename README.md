# ASTROC2M | Security-First Web3 Education and Digital Utility Ecosystem on Polygon

**Status:** Security Review Active — Public dApp Temporarily Paused  
**Network:** Polygon Mainnet  
**Purpose:** Web3 education, safer wallet interaction, public verification, NFT utility access, and community tools.

ASTROC2M is a Web3 education and digital utility ecosystem built on Polygon.

The project is structured around security-first onboarding, transparent verification, documentation, NFT-based access, internal ecosystem utility, and community interaction tools.

ASTROC2M is not positioned as a speculative investment product and does not promise profit, yield, dividends, financial returns, or guaranteed future value.

## Security Notice

ASTROC2M will never request:

- seed phrases
- private keys
- recovery phrases
- wallet passwords
- unrelated signatures
- unlimited token approvals

Only connect wallets through the official dApp:

https://app.astroc2m.com

Always verify official links through the ASTROC2M Trust Center before interacting with any Web3 feature:

https://astroc2m.space/trust-center/

---

## Current Safety Status

The public dApp has been temporarily paused while the wallet connection and minting flow are being reviewed and improved.

A safer V2 architecture has been prepared with:

- no Web3 loaded before explicit user confirmation
- no WalletConnect in the initial flow
- no Web3Modal global provider
- no setApprovalForAll in the mint flow
- official domains and contracts displayed before wallet connection
- wallet connection only after explicit user confirmation
